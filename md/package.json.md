版本号有前缀（~或^）
=
1. 波浪符号（~）：  
  会更新到中间的那位数字中最新的版本。  
  ~1.15.2，匹配1.15.x的最新版本  
  如果出了一个新的版本为1.16.0，则不会自动升级。  
  波浪符号是曾经npm安装时候的默认符号，现在已经变为了插入符号。

2. 插入符号（^）：  
  他将会把当前库的版本更新到第一位数字中最新的版本。  
  3.4，匹配3.x.x中最新的版本，但是他不会自动更新到4.0.0。  

3. 波浪符号和插入符号可以同时使用，例如：
```json
  "dependencies": {
    "lodash": "^1.0.0 || ~1.2.1",
    "react": "^16.0.0",
    "vue": "^2.5.2"
  }
  ```
